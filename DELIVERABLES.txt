================================================================================
POLYNOMIAL STRING FORMAT
================================================================================

Format: "c1*u^d1*x^e1 + c2*u^d2*x^e2 + ..."

Examples:
  "1"                        → constant 1
  "u^2"                      → u^2 (implicit coeff=1)
  "2*u*x"                    → 2*u*x (implicit powers=1)
  "1*u^2 + 1*x^2"            → u^2 + x^2
  "u^3 + x^3"                → u^3 + x^3 (short form)
  "1*u^4 + 2*u^2*x + 1*x^4"  → u^4 + 2*u^2*x + x^4
  "5*u^3 - 2*u*x + x^2"      → 5*u^3 - 2*u*x + x^2 (with subtraction)

Parser: Strips whitespace, handles +/- signs, extracts coefficients/exponents
